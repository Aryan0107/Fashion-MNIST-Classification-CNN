# Fashion MNIST Classification using CNN

This is a beginner-friendly Deep Learning project that classifies fashion items using a Convolutional Neural Network.

## Project Overview

The goal of this project is to build a CNN model that can classify grayscale images of fashion items into different categories.

The project uses the Fashion MNIST dataset, which contains images of clothing items such as T-shirts, trousers, shoes, bags, and more.

## Dataset

The Fashion MNIST dataset contains:

- 60,000 training images
- 10,000 testing images
- Image size: 28 x 28 pixels
- 10 classes of fashion items

## Classes

The model classifies images into the following categories:

- T-shirt/top
- Trouser
- Pullover
- Dress
- Coat
- Sandal
- Shirt
- Sneaker
- Bag
- Ankle boot

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Model Architecture

The CNN model contains:

- Convolutional layers
- MaxPooling layers
- Flatten layer
- Dense hidden layer
- Output layer with Softmax activation

## Results

The model achieves good accuracy on the Fashion MNIST test dataset.
